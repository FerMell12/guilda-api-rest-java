# Guilda dos Caçadores - API REST

Projeto desenvolvido para a disciplina de Programação.

## Objetivo

Desenvolver uma API REST em Java para gerenciamento de uma Guilda de Caçadores de Monstros.

A API permitirá o gerenciamento de:

- Caçadores
- Monstros
- Missões

## Tecnologias

- Java 21
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Git
- GitHub
- Postman

## Funcionalidades

### Caçadores

- Criar
- Listar
- Atualizar
- Excluir

### Monstros

- Criar
- Listar
- Atualizar
- Excluir

### Missões

- Criar
- Listar
- Atualizar
- Excluir

## Recursos Extras

- Validação de dados
- Tratamento de erros
- Paginação
- Filtros
- Relacionamentos entre entidades
